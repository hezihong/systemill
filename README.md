# systemill

## 项目描述：
```
病理信息信息管理系统是为医护人员专门开发的一套管理系统。本系统在功能方面主要包括：用户信息管理，患者个人信息管理，病理信息管理。
该系统整体构架简单，用户容易上手，操作快捷、方便，性能高效、强大。此系统全中文操作界面，使用易懂、易会。
```
## 技术描述：
```
1.运用了前后端分离的开发思想，用Vue.js实现前端页面，用Node.js编写后台；
2.运用Express应用框架接收请求消息并返回响应消息；
3.利用session实现用户的登录功能，并使用md5加密技术对用户的密码进行加密；
4.运用mysql数据库对数据进行增删改查等操作；
5.运用fs模块与formidable模块实现文件的上传功能；
```
